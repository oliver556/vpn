# Loon

Loon 是一个集网络代理、网络调试、网络改写等一系列功能于一体的网络工具。

## 获取 Loon

目前 Loon 已经被政府要求从中国大陆区的 App Store 移除。请切换至非中国大陆地区（如美国、香港、日本地区），再购买下载。

> [!TIP]
> 如果你想注册自己的非大陆地区的 App Store 账号，请可以参考如下教程: [港区 Apple ID 注册流程]

## 下载 Loon

打开 "App Store" 应用 > 我的头像 > 滑动到页面底部注销您的 Apple ID > 登陆非中国大陆区 Apple ID >搜索 [Loon](https://apps.apple.com/us/app/loon/id1373567447) 并安装
